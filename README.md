#  Grand Stay Hotel Website

A responsive multi-page hotel website built using **HTML, CSS, and JavaScript**. This project demonstrates modern frontend web development concepts including responsive design, navigation, image gallery, booking form, and a room cost calculator.

## Features

- Responsive navigation bar
- Modern homepage with hero section
- About Us page
- Rooms page with pricing
- Booking cost calculator
- Hotel image gallery
- Contact & booking form
- Mobile-friendly design
- Clean and organized code structure

##  Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Pages

- **Home** – Welcome page with featured room and hotel highlights.
- **About** – Information about the hotel and its services.
- **Rooms** – Room types, pricing, and booking cost calculator.
- **Gallery** – Hotel and room image gallery.
- **Contact** – Contact details and booking enquiry form.

## How to Run

1. Download or clone this repository.

```bash
git clone https://github.com/your-username/grand-stay-hotel.git
```

2. Open the project folder.
